The Weekly Toolbox (WN30)

A week wouldn’t go by without stumbling upon a new tool or service. Instead of committing to memory why not commit to Git and create a semipermanent record? Not limited to opensource, I will also feature commercial products occasionally found rummaging through product placements or news.

Occasionally posted these on Twitter but organization and recall are horrible. Would also like to write a few words about the tool and any experience with it. You can typically find similar content in email newsletters. I can probably do 1-3 tools per week. I’m fine to be the only one reading my posts but it might be interesting for somebody else.

This is the first „The Weekly Toolbox“, week number 30 of 2021. Only one for today, yubikey-agent.

yubikey-agent

It’s possible to use the smart card functionality of recent Yubikey models to store SSH keys. This is another more secure option to the built-in FIDO U2F feature of OpenSSH. Be aware though, not all Yubikeys have smart card functionality. I think Yubikey 4 & 5 have it. You can think of yubikey-agent as a ssh-agent replacement since you’re going to pass the SSH agent socket from yubikey-agent. The project page at Github shows a screenshot of it in action.

Compiling is a cinch since the program is written in Go but linking statically is not straightforward since it links to a C library. I have installed pcsc-lite-devel to get the headers and pickup the runtime dependency. This dependency has different names across distros. Works well with GUI pinentry alternatives on macOS and Linux. Needs some TTY hacking similar to gpg-agent + SSH for console pinentry. On Linux, you also have to muck a bit with Polkit and its JavaScript rules. If you prefer a precompiled binary, you can install it via Homebrew on macOS or just fetch the corresponding binary from their new asset hosting, Github Packages. Just found out they moved from JFrog Bintray recently. Looks like it was shut down.

Written by the Go Security lead, BSD 3-Clause licensed, and it’s for SSH. What’s not to like? Check it out at github.com/FiloSottile/yubikey-agent. Also added to github.com/tongson/rice-cooker.

Looking forward to a great weekend.
